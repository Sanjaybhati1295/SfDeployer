Privacy Policy

Last updated: March 2026

This Chrome extension connects to Salesforce organizations authorized by the user through OAuth.

Information We Access

The extension may access Salesforce metadata such as Apex classes, objects, fields, flows, and deployment configuration required to perform metadata retrieval and deployments.

Authentication

Authentication is performed using Salesforce OAuth. The extension does not collect or store user passwords.

Data Storage

The extension may store limited information locally using Chrome storage such as connected org information, selected metadata components, and user preferences.

Data Sharing

No personal data is sold or shared with third parties.

Security

All communication between the extension, backend services, and Salesforce APIs occurs over secure HTTPS connections.

Contact

For questions, contact: support@deployforce.com
